## **알바비 계산기**
- 3.3% 세금을 제외한 실 수령 금액 계산 위한 간단한 코드
- 일한 날짜를 입력하면 일한 시간을 계산해주어 출력(근무태도 제출용)
- 시급은 2024년 최저시급 9820원으로 설정. 야간시급은 1.5배
- 일한 시간은 작성자가 일하는 17:00~23:00으로 설정. 야간시간은 22:00~23:00
